# Synthetic Product Docs for Swagger Petstore

This folder contains synthetic English product documentation that extends the public Swagger Petstore OpenAPI seed dataset.

The documents are intentionally small, structured, and safe for a public portfolio repository. They are not copied from any private workplace material. Each document references OpenAPI tags, paths, operation IDs, schemas, and security schemes from `raw_data/openapi.yaml`.

Current artifact types:

- `user_story`
- `acceptance_criteria`
- `incident_note`

These docs are raw source data for RAG ingestion experiments. They should be parsed alongside `raw_data/openapi.yaml` to test retrieval across API contracts and product-style documentation.
